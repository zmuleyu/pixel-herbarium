## 1. 物候时序（花期顺序）

| 顺序 | 花卉 | 特征 |
|------|------|------|
| 最早 | 梅花 | 叶前开花，花梗极短，花贴枝，单瓣/重瓣均有 |
| 第2 | 玉兰 | 先花后叶，大型白色/紫色钟形，无花梗直立枝头 |
| 第3 | 早樱（山樱等） | 花叶同放，花梗长、下垂，花瓣顶端有缺口 |
| 第4 | 中樱（染井吉野） | 先花后叶，浅粉色，日本最常见赏樱品种 |
| 第5 | 桃花 | 花单生，无花梗或极短，颜色鲜艳（深粉/红/白） |
| 第6 | 海棠 | 伞房花序，花梗长，下垂，深粉变粉白渐变 |
| 最晚 | 晚樱（关山/郁金） | 重瓣，花期最晚，郁金樱独特黄绿色 |

**PH 应用**：`src/constants/plants.ts` 的 bloomStart/bloomEnd 字段可据此校验，重点验证梅→玉兰→早樱→中樱→桃→海棠的时序先后关系。

## 2. "六姐妹"辨识系统

> 早春最易混淆的六种花卉，图鉴原文以此为核心框架

| 花卉 | 辨识关键点 | 常见误识 |
|------|-----------|---------|
| **梅花** | 短枝有刺，花单生，花梗极短，萼片反折 | 误认为早樱 |
| **桃花** | 花单生，花萼5片，有绒毛，叶细长披针形 | 误认为海棠 |
| **樱花** | 花梗长（下垂），花瓣顶端有"V形缺口" | 误认为梅花/桃花 |
| **海棠** | 花多朵聚生（伞房序），花梗长，叶卵圆形 | 误认为桃花 |
| **玉兰** | 花大，6-9瓣，花叶不同放，单生枝顶 | 误认为白色樱花 |
| **棣棠** | 纯黄色5瓣/重瓣，叶片有明显锯齿 | 误认为迎春/连翘 |

### 快速口诀（原文提炼）
- **梅**：花贴枝，萼反折，冬末开
- **桃**：叶细长，花鲜艳，单朵生
- **樱**：梗长垂，瓣有缺，最好辨
- **海棠**：花成束，颜色渐，叶圆润
- **玉兰**：花朵大，先开花，不见叶

## 3. 四小黄花辨识

> 早春四种易混淆的黄色花卉

| 花卉 | 关键特征 |
|------|---------|
| **迎春花** | 6瓣，绿色枝条拱垂，花与叶同放 |
| **连翘** | 4瓣，枝条直立/拱形，花先叶开 |
| **棣棠** | 5瓣/重瓣，叶有锯齿，枝细绿色 |
| **黄素馨** | 类迎春但叶三出复叶，更大朵 |

## 4. 樱花三期系统

| 期别 | 代表品种 | 特征 | 花期（日本） |
|------|---------|------|------------|
| 早樱 | 山樱、寒绯樱 | 花叶同放，颜色较深粉/紫粉 | 3月上旬 |
| 中樱 | 染井吉野（ソメイヨシノ）、大岛樱 | 先花后叶，浅粉白，最常见 | 3月下旬-4月上旬 |
| 晚樱 | 关山（カンザン）、郁金（ウコン） | 重瓣，郁金为黄绿色独特 | 4月中下旬 |

**PH 应用**：PH 若含樱花品种（如 ソメイヨシノ），应归属中樱品种组；花期 3/25-4/10（东京）。

## 5. PH 设计借鉴点

### 5.1 知识卡片（翻卡第三面）

图鉴文章用"对比表格 + 关键特征突出"的方式展示辨识信息，这正是 PH 翻卡知识面应该的格式：

```
┌─────────────────────────────┐
│  [花卉名]  ·  [稀有度★★]     │
│                             │
│  🌸 开花时序：春季第N位      │
│  📍 辨识关键：花梗长，瓣有缺  │
│  💡 花言葉：○○○○            │
│                             │
│  [与相似花种的对比小图]       │
└─────────────────────────────┘
```

### 5.2 "辨识挑战"玩法（待设计）

基于"六姐妹"混淆现象，可设计 PH 专属玩法：
- 显示一张花朵特写图，用户从 4 个选项中辨识正确花名
- 答对解锁知识卡片附加内容（辨识口诀、物候排序等）
- 适合作为 **Phase 4 / 内容扩展** 的轻量玩法
