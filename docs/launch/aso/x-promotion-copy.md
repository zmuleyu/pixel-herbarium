## 九、文案使用注意事项

### 9.1 语气原则
- X 平台用户信息流密集，**短句优先，首句必须抓眼球**
- 善用**换行留白**制造节奏感与情绪停顿
- **避免硬促销感**，以情感叙事或提问引导互动
- 话题标签控制在 **3–5 个**，优先使用日语热门赏樱话题

### 9.2 互动策略
- 每周至少 1 条**明确互动引导**（评论、转发、截图晒单）
- UGC 话题建议设置**固定标签**，便于聚合用户内容
- 图鉴连载推文建议配合**高质量实景图** + **APP 截图**双图展示

### 9.3 数据占位符说明
- `[地点名]` → 替换为具体赏樱地点名称（日语优先）
- `[N]` → 替换为具体数字
- `[APP链接]` / `[APP下载链接]` → 替换为正式发布后的下载链接
- `[活动详情链接]` / `[キャンペーンリンク]` → 替换为对应活动落地页
- `[好友名]` / `[友だち名]` → 系统自动读取用户好友昵称
