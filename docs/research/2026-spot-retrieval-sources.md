## 六、检索实操工作流

### 第一步：确认目标地区开花状态

```
1. → tenki.jp/sakura/expectation/    查最新满开预想日
2. → 気象庁官网                      核实是否已正式开花观测
3. → weathernews.jp/sakura/          查实时用户上报开花状况
```

### 第二步：筛选热门景点

```
1. → hanami.walkerplus.com/ranking/  获取各地区人气排名
2. → sakuranokai.or.jp              核对是否为100选名所
3. → tour.ne.jp/matome/articles/hanami/  参考编辑精选70处
```

### 第三步：深挖景点详情

```
1. → 各地官方观光网站（见下方链接表）  获取权威活动/交通信息
2. → tenki.jp 地点详情页               夜樱/祭り活动确认
3. → Google Maps 手动搜索景点名称       参考评分与评价数量
```

### 第四步：追踪实时现场情报

```
1. → X 搜索「[景点名] 桜 [日期]」      实时打卡内容
2. → Instagram #[景点名]+桜             视觉热度判断
3. → LINE OpenChat 相关地域群组         本地用户第一手情报
```
