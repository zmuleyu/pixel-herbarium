# 🌸 Pixel Herbarium「赏樱打卡」玩法推广文案集

> **功能概述**：用户前往真实的樱花观赏地点，在 APP 内完成地点打卡，解锁对应地点的像素樱花图鉴，并可为拍摄的照片添加地点 Logo 后分享。
> 
> **文案定位**：情感驱动 × 收集养成 × 线下打卡仪式感
> 
> **目标受众**：日本及亚洲都市女性，20–35 岁，热爱自然与生活记录

## 四、文案使用注意事项

### 4.1 语气原则
- **日语渠道（LINE / X / Instagram 日文账号）**：语气温和，保持"物の哀れ"的情感底色，避免过度促销感
- **中文渠道（小红书 / 抖音）**：干货感 + 情感感并重，标题需含有明确的利益点或悬念

### 4.2 图片配合建议
- 所有社交推广文案建议配合**高质量实景樱花摄影** + **APP 像素图鉴截图**双图呈现
- 避免使用人工合成感过强的樱花图片，优先使用真实拍摄素材

### 4.3 数据占位符说明
- `[地点名]` → 替换为具体赏樱地点名称（日语/中文）
- `[N]` → 替换为具体数字
- `[APP链接]` → 替换为正式发布后的下载链接
- `[活动链接]` → 替换为对应活动落地页
