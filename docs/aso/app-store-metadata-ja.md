## 説明文（4000文字以内）

```
花に出会ったら、カメラを向けてみてください。

花図鉑（はなずかん）は、日本の野花や街角の花を撮影して、ピクセルアートに変換する花図鑑アプリです。AIが植物を識別し、あなたの写真を温かみのあるピクセルアートに生まれ変わらせます。

花言葉と一緒に、あなただけの花図鑑を作りましょう。


◆ 撮って、識別して、ピクセルアートに

見つけた花をカメラで撮影するだけ。AIが植物の種類を識別し、あなたの写真をもとにオリジナルのピクセルアートを生成します。

既存の植物識別アプリとは違い、花図鉑では「識別」がゴールではありません。あなたの発見が、ひとつの小さなアート作品に変わる——それが花図鉑の楽しみです。


◆ 花言葉で、花をもっと深く知る

日本には古くから伝わる花言葉（はなことば）の文化があります。花図鉑では、発見した花の花言葉と西洋の花言葉をカードの表裏で楽しめます。

タップするとカードがくるりと裏返り、もうひとつの意味が現れる——小さな発見の連続です。


◆ 季節とともに移ろう図鑑

春は桜やチューリップ、夏は向日葵や朝顔。季節ごとに出会える花が変わり、図鑑は少しずつ色づいていきます。

花開き → 見頃 → 花散り。三段階の季節インジケーターが、今この瞬間の花の状態を教えてくれます。

★★★の限定植物は、決まった時期にだけ発見できる特別な存在。来年また咲きます——焦らず、自然のリズムで楽しんでください。


◆ 同城マップで、花の地図を描く

あなたの発見がGPS付きで地図に記録されます。東京、大阪、京都——同じ街に住む人たちがどこでどんな花を見つけたか、ヒートマップで確認できます。

桜の名所、紫陽花の穴場、秘密のスミレスポット。みんなの発見が重なって、街の花の地図が出来上がっていきます。


◆ 花束を贈る

お気に入りの花を3〜5種類選んで、友達にピクセルアートの花束を贈りましょう。花言葉を添えたメッセージカード付き。

「この花をあなたに贈ります」——花図鉑だけのギフト機能です。


◆ こんな方におすすめ

- 散歩中に見つけた花の名前を知りたい方
- ピクセルアートやドット絵が好きな方
- 花言葉や植物の文化に興味がある方
- コレクション型のアプリが好きな方
- 季節の移ろいを感じながら暮らしたい方
- 友達に素敵なギフトを贈りたい方


◆ 主な機能

・AI植物識別 — カメラで撮影するだけ
・ピクセルアート変換 — あなたの写真がアートに
・花言葉データベース — 日本の花言葉 + 西洋の花言葉
・花図鑑コレクション — 120種の植物を収集
・季節のリズム — 花開き → 見頃 → 花散り
・同城マップ — GPS付き発見記録 + ヒートマップ
・花束ギフト — ピクセルアート花束を友達に
・シェアポスター — Instagram Stories / LINE カード


◆ 無料で始められます

毎月5回まで無料で植物を識別できます。まずは身近な花から始めてみてください。


◆ プライバシー

・発見の位置情報は約100m曖昧化して記録されます
・地図への公開はオン/オフ切り替え可能
・データのエクスポートとアカウント削除に対応


花図鉑は、あなたの日常に小さな発見と温かさを添えるアプリです。

今日はどんな花に出会えるでしょうか。
```
