### A-3　国土地理院（GSI）— 日本国家地理空间信息局

| 项目 | 内容 |
|---|---|
| **简介** | 日本政府官方地理信息机构，提供全国地名、坐标、地图瓦片数据 |
| **数据内容** | 地名检索（地址→坐标）、地图瓦片、标高数据 |
| **授权状态** | 🟢 政府开放数据，商业使用基本可行，需标注出处 |
| **主页URL** | https://www.gsi.go.jp/ |
| **地名检索API** | `https://msearch.gsi.go.jp/address-search/AddressSearch?q={地点名}` |
| **地图瓦片** | `https://cyberjapandata.gsi.go.jp/xyz/std/{z}/{x}/{y}.png` |
| **地理院地图** | https://maps.gsi.go.jp/ |
| **获取方式** | 地名检索 API（免费，无需 Key，返回 GeoJSON 格式坐标） |

```python
# 示例：通过地点名获取坐标
import requests, urllib.parse

def get_coords(place_name):
    url = "https://msearch.gsi.go.jp/address-search/AddressSearch"
    r = requests.get(url, params={"q": place_name})
    data = r.json()
    if data:
        coords = data[0]["geometry"]["coordinates"]
        return {"lng": coords[0], "lat": coords[1]}
    return None

print(get_coords("上野恩賜公園"))
# → {"lng": 139.7714, "lat": 35.7141}
```

| **使用建议** | ✅ 作为坐标验证工具：已有地点名称时，用此 API 批量获取精确坐标 |
| **注意事项** | 官方说明未设具体请求次数限制，但建议请求间隔 ≥1 秒；精度不如 Google Maps，偏远地点可能查询失败 |
