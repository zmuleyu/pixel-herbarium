# App Store 评论回复规范

> Adult Kawaii 调性 · 日语为主 · 版本 v1.0 · 2026-03-20

---

## 回复原则

1. **温柔不焦虑**：用自然季节隐喻，禁止防御性语气
2. **具体不敷衍**：回复问题时说明具体改进或解决方案
3. **48小时内回复**：特别是 1-2 星评论
4. **不超过 3 行**：简洁，在 App Store 界面不截断

---

## 常见问题回复

### "写真が保存されない"（照片无法保存）
```
ご不便をおかけして申し訳ありません。
設定→プライバシー→写真でアプリへのアクセスをご確認ください。
それでも解決しない場合はお知らせください🌿
```

### "起動しない / クラッシュする"（启动崩溃）
```
申し訳ございません。アプリを一度削除して再インストールをお試しください。
改善されない場合は [support email] までご連絡ください。
次回アップデートでも対応いたします🌱
```

### "対応花が少ない"（花种太少）
```
ご意見ありがとうございます。
現在[X]種類の花に対応しており、季節ごとに追加予定です🌸
どんな花を追加してほしいか、ぜひ教えてください！
```

### "英語のみにしてほしい"（要求英文）
```
Thank you for your feedback!
English is already supported — please check Settings → Language.
We hope you enjoy the blooming seasons! 🌸
```

---

## 禁止用语

- ❌ 「申し訳ございません」× 3回以上（过度道歉）
- ❌ 「検討します」无具体时间（空洞承诺）
- ❌ 「他のユーザーはうまく動いています」（防御性）
- ❌ 倒计时/紧迫感语言（违反 Adult Kawaii 调性）
- ❌ 英文回复日文评论（除非评论本身是英文）

---

## 回复频率

- 每周检查一次新评论（集中处理）
- 1-2星：48小时内回复
- 3星以上：7天内回复
- 季节高峰期（3月/11月）：每日检查

*数据来源：`scrapling-mcp` appstore_reviews 工具 + ph-daily cron*
