## 推送文案框架（三阶段 FOMO）

每季最多 **3次** 推送，用温柔自然隐喻，禁止倒计时/红色/"错过了"。

### 阶段1：花開き（开始）
```
今年も[花名]の季節がやってきました🌸
[景点名]で最初の[花名]が咲き始めています。
お散歩がてら、ぜひ記録してみてください。
```

### 阶段2：見頃（盛开）
```
[地域]の[花名]が見頃を迎えています✨
今週末が最高のタイミングかもしれません。
あなたの花図鉑に加えてみませんか？
```

### 阶段3：花散り（结束）
```
[花名]の季節も、もうすぐ終わりを迎えます🍃
来年また咲きます——でも今年の記録は今だけ。
[X]か所まだ未訪問のスポットがあります。
```

> ⚠️ 阶段3禁用："もうすぐ終わり！" / "今しかない！" / 红色感叹号
